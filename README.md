# CSGO Buddy
Makes your favorite ESport title a little bit better. 

## Prerequisite
- Node.js installed
- CSGO Launch option: 
```
-netconport 1337
```

## Run
1. Start CSGO
2. Run following in CMD
```
npx csgo-buddy
```

Features:
 - Chat translation to English

## Warning
This buddy will mess with your developer echos.
